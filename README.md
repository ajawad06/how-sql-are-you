# 🪄 How SQL Are You?

**How SQL Are You?** is a quiz web app that tests your SQL knowledge with interactive questions and scores your responses.

## 🚀 Features
- Multiple‑choice SQL questions
- Real‑time scoring
- Fun visual feedback
- The user is classified into one of 5 levels, based on the number of correct answers.

## 👷 Built With
- **Lovable.dev** – AI‑driven full‑stack app generation
- **React** – Frontend UI

## 🎮 Usage
1. Visit the app: https://how-sql-are-you.lovable.app/
2. Start the quiz and answer the SQL questions.
3. View your final SQL savvy score.

## ⛏️ Development
This project was created by providing Lovable a natural‑language description of the desired quiz functionality.
